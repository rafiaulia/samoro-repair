Pertanyaan mengenai Ras Repair

Kapan waktu operasional Ras Repair? +
Ras Repair buka setiap hari dari pukul 09.00 – 18.00 WIB.
Apakah Ras Repair melayani pemasangan Windows bajakan? +
Tidak, kami hanya melayani pemasangan Windows legal. Jika lisensi tidak tersedia, tersedia opsi Linux (Linux Mint).
Apa saja yang termasuk dalam layanan install ulang? +
Layanan meliputi install ulang sistem operasi, driver, software legal dasar, dan konfigurasi awal perangkat.
Apakah data saya akan dibackup sebelum install ulang? +
Backup dilakukan hanya atas persetujuan pelanggan. Pelanggan bertanggung jawab memastikan data penting sudah dibackup.
Apa garansi yang diberikan Ras Repair? +
Garansi berlaku untuk sistem operasi dan aplikasi yang diinstal. Jika terjadi masalah terkait layanan, kami menyediakan service ulang atau garansi uang kembali.
Berapa lama biasanya proses install ulang selesai? +
Proses biasanya memakan waktu 1 sampai 3 hari kerja, tergantung kondisi perangkat dan kendala teknis yang ditemukan.
Apakah Ras Repair menyediakan nota untuk setiap transaksi? +
Ya, nota digital dan fisik tersedia untuk setiap transaksi (opsional sesuai permintaan pelanggan).
Apakah Ras Repair memasang software ilegal seperti Office crack atau Adobe crack? +
Tidak. Semua software yang kami instal bersifat legal dan gratis. Software bajakan tidak kami layani.
Apakah Ras Repair menjaga kerahasiaan data pelanggan? +
Ya, semua data pelanggan dijaga kerahasiaannya dan hanya digunakan untuk keperluan proses service.
Apakah Ras Repair bisa menolak layanan? +
Ras Repair berhak menolak layanan apabila perangkat terindikasi digunakan untuk aktivitas ilegal atau tidak memenuhi syarat layanan.
